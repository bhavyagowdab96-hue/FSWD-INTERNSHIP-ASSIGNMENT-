What Happens When You Open a Website?

  1. Enter URL: User types a website link in the browser.
  2. DNS Lookup: URL is converted into an IP address.
  3. TCP Handshake: Browser creates a connection with the server.
  4. HTTPS Handshake: A secure encrypted channel is established.
  5. HTTP Request: Browser sends a GET request to server.
  6. Server Processing: Server retrieves webpage files or database data.
  7. HTTP Response: Server sends HTML, CSS, JS, images, etc.
  8. Rendering: Browser builds DOM and displays the webpage.
  9. Additional Requests: Browser loads images, CSS, scripts, icons.
